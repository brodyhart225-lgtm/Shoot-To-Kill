Shoot to Kill

Features:

Wave-based enemy system

Multiple weapons (with more planned)

Player health and survival mechanics

Fast-paced gameplay loop

Simple controls, easy to pick up

--------------------------

Controls:

Mouse: Aim

Mouse button (any): shoot (better for mouse)

S: shoot (better for trackpad)

R:	Reload

Space: Quit current mode

--------------------------

Installation:

Clone the repository:

git clone https://github.com/yourusername/shoot-to-kill.git

cd shoot-to-kill

--------------------------

Install dependencies:

pip install pygame

--------------------------

Running the game:

python3 main.py

--------------------------

Requirements:

Python 3.x

Pygame
